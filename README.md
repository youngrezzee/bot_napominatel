# Telegram бот-напоминатель

Бот принимает сообщения в формате:

```text
16.03.2026 18:30 Встреча с командой
```

После этого он сохраняет событие и отправляет напоминания в тот же чат:

- за 1 день
- за 3 часа
- за 1 час

## Что умеет

- принимать события обычным текстом
- хранить события в SQLite (`events.db`)
- восстанавливать напоминания после перезапуска
- показывать список событий командой `/list`
- удалять событие командой `/delete <id>`

## Запуск

1. Установить Python 3.11+.
2. Установить зависимости:

```bash
pip install -r requirements.txt
```

3. Создать Telegram-бота через `@BotFather` и получить токен.
4. Задать переменные окружения:

```bash
set TELEGRAM_BOT_TOKEN=ваш_токен
set BOT_TIMEZONE=Europe/Moscow
```

5. Запустить:

```bash
python bot.py
```

## Важный момент для чатов

Если бот должен реагировать на обычные сообщения в групповом чате, у него нужно отключить privacy mode через `@BotFather`, иначе он будет видеть в основном только команды.

## Формат сообщений

Поддерживается формат:

```text
ДД.ММ.ГГГГ ЧЧ:ММ Название события
```

Пример:

```text
21.03.2026 09:00 Поезд в аэропорт
```
